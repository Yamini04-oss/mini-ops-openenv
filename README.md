## Task Descriptions

### Task 1 вЂ” Email Classification (Easy)

The agent receives a raw email (subject + body) and must classify it as either `important` or `ignore`.

**Input:** Plain text email  
**Expected output:** `"important"` or `"ignore"` (single word, case-insensitive)  
**Grading:** 1.0 for exact match, 0.5 if the correct label appears anywhere in the answer

---

### Task 2 вЂ” Task Prioritization (Medium)

The agent receives a list of tasks with numeric priority scores and must return them sorted from highest to lowest priority.

**Input:** JSON list of `{name, priority}` objects  
**Expected output:** JSON array of task names in descending priority order  
**Grading:** 1.0 for perfect order; 0.5вЂ“1.0 for correct items in wrong order; 0.0вЂ“0.2 for wrong items

---

### Task 3 вЂ” Data Cleaning (Hard)

The agent receives a list of records where the `value` field may be a number, `null`, or a non-numeric string. It must sum only the valid numeric values.

**Input:** JSON list of `{id, value}` records  
**Expected output:** A single numeric value (e.g. `42.5`)  
**Grading:** 1.0 for exact match; 0.5 within В±5; 0.2 within В±20; 0.0 otherwise

## Setup & Usage

### Option 1 вЂ” Run locally (no Docker)

```bash
# 1. Install dependencies
pip install -e .

# 2. Start the server
uvicorn mini_ops_env.server.app:app --host 0.0.0.0 --port 7860

# 3. In another terminal, run the baseline
export HF_TOKEN=hf_your_token_here
python baseline.py
```

### Option 2 вЂ” Docker

```bash
# Build
docker build -t mini-ops-env .

# Run
docker run -p 7860:7860 mini-ops-env

# Baseline (against running server or locally)
export HF_TOKEN=hf_your_token_here
python baseline.py
```

### Option 3 вЂ” Hugging Face Spaces

Deploy this repository as a **Docker Space** on Hugging Face. The `openenv.yaml` is included for automatic discovery.
